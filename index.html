Surf Video and Forecasts Dataset

Create a dataset of videos of surf spots along with hourly forecasts.

I used Surfline's forecast API to periodically (approximately daily) update a database with the surf forecast for several surf spots. I also scrape Surfline using Selenium to find the links to the CDN server for rewind clips of the surf cams. These are 10 minute videos taken back to back continuously. Once the links to the CDN server have been catalogued along with their timestamp, I select one video an hour whilst there is sunlight, then download and process the video. Processing is done with FFMPEG and is simply cutting the video to 60s and compressing it.

Time Machine Surf Cam

Surf cam footage (for forecast conditions) for days in the future.

This project is currently incomplete as it needs a larger dataset. Currently, there is a primitive nearest neighbor implementation. This builds on the surf video and forecast dataset to deliver videos of the nearest datetime.

Symptom Log

Django website to track and monitor patient symptoms for clinicians to review.

This is a django website which allows clinicians to create custom forms (made up of different question styles) for patients to fill in. This allows patient's symptoms to be monitored and recorded so that doctors can easily view how a patient's condition is progressing and view patterns with symptoms and treatments.
This project is currently under development and charts for visualisation are not yet implemented.

Symptom Visualisation Dashboard (R Shiny)

Custom dashboard to display pain severity for patient and medical device activity.

This project was the motivation and starting point of Symptom Log. I initially built this project for my dad, whilst he was in hospital. The datasource was a google sheets table which was updated by a google forms. It tracked pain severity of different body parts, overall wellbeing and crucially whether a medical device (neuro stimulator) was turned on or off. This dashboard displayed this data in an R shiny app and was deployed so doctors could see if this medical device was working. The github repo only conains an anonymized (synthetic, unlabelled data) version of this for privacy.

Analysis of Global Buoy Data

I use Google BigQuery to investigate the NOAA ICOADS database. I explore interesting variables and plot global buoy locations. I then focus on a specific buoy and calculate moving averages of various variables with SQL window functions.

Hawaii Waveheight Prediction

Predicts waveheights in Oahu, Hawaii with offshore buoy data.

I use an XGBoost model to use offshore buoy data to predict max recorded waveheights on west of Oahu, Hawaii.

Durham Police Dashboard

Interactive shiny dashboard to display crime data from the UK Police API.

Interactive dashboard to show UK Police data for Durham. You can select a constituency and a month period and it shows a barchart for categories of crimes along with an interactive map showing the locations. I built this app with r and shiny whilst at university in Durham, and it was interesting to see the crime show up on the map from when somebody stole my numberplates!
